# Checklist de Publicacao do PWA

## Objetivo

Garantir que a versao corrigida chegue ao site e ao celular antes de concluir um atendimento.

## Checklist

- Confirmar arquivos alterados.
- Rodar build local.
- Informar comandos de git ao usuario.
- Confirmar push realizado.
- Confirmar deploy finalizado na Vercel.
- Testar hard refresh no navegador.
- Testar reinstalacao do PWA quando necessario.
- Confirmar que a tela publicada reflete a mudanca esperada.
