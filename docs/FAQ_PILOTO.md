# FAQ do Piloto Gratuito

## O piloto e pago?
Nao. Nesta fase e uma demonstracao gratuita para validar melhorias.

## Quem pode entrar?
Somente participantes convidados pelo administrador.

## Como envio sugestoes?
Pela aba **Enviar Feedback** dentro do sistema.

## Meus dados estao protegidos?
Sim. O sistema tem autenticacao, trilha de auditoria e funcoes LGPD.

## O que acontece ao final do piloto?
Os feedbacks sao priorizados para evolucao antes da comercializacao.
