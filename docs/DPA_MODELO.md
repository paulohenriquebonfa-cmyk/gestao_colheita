# DPA - Acordo de Tratamento de Dados (Modelo Inicial)

## Partes
- Controlador: cliente contratante (fazenda/cooperativa/empresa).
- Operador: fornecedor do sistema.

## Objeto
Tratamento de dados pessoais estritamente para operacao do sistema de gestao de colheita.

## Obrigacoes do Operador
- Tratar dados conforme instrucao do Controlador.
- Aplicar medidas tecnicas e organizacionais razoaveis de seguranca.
- Apoiar atendimento aos direitos dos titulares.
- Comunicar incidente de seguranca relevante sem demora indevida.

## Suboperadores
- Provedor de nuvem/banco utilizado no ambiente do cliente.

## Encerramento
- Em caso de termino, dados podem ser devolvidos/exportados e/ou excluidos conforme instrucao do Controlador.

> Documento modelo inicial. Recomenda-se revisao juridica antes da comercializacao em escala.
