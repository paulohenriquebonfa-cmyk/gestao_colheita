# Termos de Uso (Modelo Inicial)

Este sistema e fornecido para gestao de colheita, armazenagem e vendas de graos.

- O cliente e responsavel pelos dados inseridos.
- O acesso deve ser feito apenas por usuarios autorizados.
- O sistema oferece operacao offline e sincronizacao com nuvem quando houver internet.
- O cliente deve manter credenciais seguras.
- O uso de backup e recomendado antes de exclusoes em massa.
- Em caso de indisponibilidade externa (provedor de nuvem/internet), o funcionamento pode ser afetado.

> Documento modelo inicial. Recomenda-se revisao juridica antes da comercializacao em escala.
